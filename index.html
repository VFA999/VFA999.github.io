Cara lihat PW history wifi dengan commandprompt


Pertama-tama buka cmd dengan run as administrator seperti gambar dibawah.

Kedua ketikkan perintah: netsh wlan show profile seperti gambar dibawah berikut... lalu enter


jika hasilnya banyak simpanan jaringan wifi seperti diatas, maka kita lanjut ke langkah selanjutnya yaitu ketikkan perintah berikut dengan benar baik spasi maupun huruf kapitalnya

netsh wlan export profile folder =C:\ key=clear

Kemudian enter

jika berhasil/sukses hasilnya akan seperti dibawah...

selanjutnya kita buka explorer lalu ke Windows (C:)

lalu cari file wifi target tadi, kemudian klik kanan open with notepad seperti gambar dibawah...
